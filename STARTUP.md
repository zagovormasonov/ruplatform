## Пошаговый запуск 🎯

### 1. Инициализация базы данных:
```bash
cd server
node init-db.js
```

### 2. Запуск сервера (порт 3001):
```bash
cd server
npm run dev
```

### 3. Запуск клиента (порт 5173):
```bash
cd client
npm run dev
```

### 4. Открыть приложение:
Перейдите в браузере на `http://localhost:5173`
